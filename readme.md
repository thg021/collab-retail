# Collab Retail
## 

[![devs](https://cldup.com/dTxpPi9lDf.thumb.png)]()


Modo simples e rápido de troca de informações dos fornecedores ligados aos varejo

- Type some Markdown on the left
- See HTML in the right
- ✨Magic ✨

## Features

- Gerenciamento de usuários;
- Envio de NFs, NFSE 
- Importação de arquivos JSON, XML etc
- Dashboards 


## Tech

- [node.js] - evented I/O for the backend
- [Express] - fast node.js network app framework [@tjholowaychuk]
- [Gulp] - the streaming build system
- [Breakdance](https://breakdance.github.io/breakdance/) - HTML
to Markdown converter
- [jQuery] - duh

And of course Dillinger itself is open source with a [public repository][dill]
 on GitHub.

## Installation

Dillinger requires [Node.js](https://nodejs.org/) v14+ to run.

Install the dependencies and devDependencies and start the server.

```sh
cd collab
npm i
```

For production environments...

```sh
npm install --production
NODE_ENV=production node app
```

## Development
Colocar aqui as instruções para subir o ambiente. 

Se precisar subir usuando o Devcontainer do vscode, precisará instalar o plugin devcontainer; 

```sh
node app
```
